## 项目描述
  * 1.此项目为一个前后台分离的招聘的SPA, 包括前端应用和后端应用
  * 2.包括用户注册/登陆, 求职者/boss列表, 实时聊天等模块
  * 3.前端: 使用React全家桶+ES6+Webpack等技术
  * 4.后端: 使用Node + express + mongodb + socketIO等技术
  * 5.采用模块化、组件化、工程化的模式开发
